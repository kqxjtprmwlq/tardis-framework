# TARDIS Framework / Marco TARDIS

This repository presents the **TARDIS Framework**, a **foundational epistemic architecture** for operating rationally under incompleteness.

It defines **conditions of operability**, not products, implementations, or concrete solutions.  
It does **not** provide software, models, or technical systems ready to use.  
Implementation and engineering are explicitly out of scope.

This repository contains the **formal specification and conceptual documentation only**.

---

**Language note**: Spanish is the reference language for this project. If there are discrepancies between Spanish and English versions, the Spanish text is authoritative.
