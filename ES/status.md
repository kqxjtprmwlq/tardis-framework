# Estado del proyecto

Este repositorio presenta un **marco epistémico fundacional**.

- La arquitectura núcleo (`ES/formal.md`) se considera **completa**.
- No se hace ninguna afirmación de verdad, corrección o validez empírica.
- El marco define **condiciones de operatividad**, no productos ni implementaciones técnicas.
- El desarrollo de sistemas concretos pertenece explícitamente al nivel de ingeniería y queda fuera del alcance de este repositorio.

El proyecto queda abierto a:
- crítica por negación explícita de axiomas,
- arquitecturas alternativas funcionales,
- implementaciones externas bajo restricciones explícitas.

En esta etapa no se planea añadir más axiomas.
